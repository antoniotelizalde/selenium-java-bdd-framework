# Selenium Java BDD Automation Framework

UI Test Automation Framework built with:

- Java
- Selenium WebDriver
- TestNG
- Cucumber (BDD)
- Maven

## Project Structure

src/main/java
- drivers
- pages
- utils

src/test/java
- runners
- stepsdefinitions
- hooks

src/test/resources
- features
- config.properties

## Test Scenario

Automation of an end-to-end user flow on Liverpool website using BDD scenarios.
